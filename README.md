# covid19-tracker

Builtwith:
- Angular v7
- [Covid-19 API](https://covid19-api.org/)


### Getting Started
To run this locally, clone the repo and use NPM to install the dependencies. (Youâ€™ll also need Node.js installed)
```bash
git clone https://github.com/januridp/covid19-tracker.git
cd covid19-tracker
npm install
```

### Development
Run:
```bash
ng serve
```
Start a dev server on [http://localhost:4200](http://localhost:4200)

### Production
To build for prod, run the following:
```bash
ng build --prod
```
